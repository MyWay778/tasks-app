# To-Do App

Task list application

### Technology stack

- Laravel (API)
- Vue3 (SPA)

### Features

- List tasks
- CRUD operations

## Development setup

```sh
composer install
```

### Compile and Hot-Reload for Development

```sh
composer run dev
```

### Useful commands

1. Fresh and populate database

```sh
php artisan migrate:fresh --seed
```

2. Run tests

```sh
php artisan test
```
